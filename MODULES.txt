For auto create a 'requirement.txt' file you can install < pipreqs >

 - pip install pipreqs

And then use the cli

 - pipreqs < path >

In this case is 

 - pipreqs ./src


And will create a 'requirement.txt' file, and remmenber must change the path in 'setup.py'
